# AI Document Assistant using RAG

## 📌 Project Overview

This project is an AI-powered document assistant that allows users to query documents and receive relevant answers. It uses a Retrieval-Augmented Generation (RAG) approach.

## 🧠 Features

- Semantic search
- Question answering
- Simple UI using Streamlit

## ⚙️ Tech Stack

- Python
- Sentence Transformers
- Streamlit
- Endee (base repository)

## ▶️ How to Run

1. Install:
   pip install sentence-transformers streamlit

2. Run:
   python -m streamlit run app/main.py

3. Open:
   http://localhost:8501

Note: Run the project locally using Streamlit to view the UI.
